Multi-Year Rainfall Excel Combiner

Upload several rainfall Excel sheets, one file per water year. The tool will rearrange all files, combine them, sort by date, and download one CSV file with two columns: Date and Rainfall_mm.



Best filename examples: 2024-2025 Mosul.xlsx, Mosul_2023_2024.xlsx, or Rainfall 2022-2023.xlsx. The first year is used as the water-year start year.

Expected sheet layout: Each Excel file should contain month headers such as ت1، ت2، ك1، ك2، شباط، اذار، نيسان، ايار، حزيران، تموز، اب، ايلول. Blank rainfall cells are automatically converted to zero.
Output: The final combined file will contain all daily records from all uploaded years, sorted from oldest date to newest date.
